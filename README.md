# Hexisyztem Skills (个人效率与日程管理方案)

## 🎯 核心目标

本项目致力于实现一套高效、灵活的个人日程与时间管理方案。

**核心设计理念：**
尽最大可能利用和整合现有的成熟工具（如滴答清单、米家等），通过编写轻量级的脚本和接口封装将它们串联起来，**避免过多的自行开发与重复造轮子**。从而构建一个以自动化、智能化为核心的个人效率系统。

---

## 🛠️ 当前已集成的能力 (Skills)

### 1. 滴答清单 (Dida365 / TickTick) 接入 ✅
基于 Dida365 OpenAPI 封装，提供了与滴答清单交互的基础能力，可以在终端或由 AI Agent 直接调用以管理日程。

**已实现的核心脚本：**
*   **创建任务/日程 (`dida365-create-schedule`)**: 允许通过命令行快速创建带有优先级、截止日期、所属项目等详细属性的新任务。
*   **查看任务层级树 (`dida365-list-tasks`)**: 以极其清晰的终端层级目录结构（📁 项目 -> 📝 任务 -> ✅/⏳ 子任务）展示账号下所有的清单与进度。

**快速启动与使用:**
调用滴答清单 API 需要先获取 Access Token。

```bash
# 1. 配置环境变量 (必填)
export DIDA365_ACCESS_TOKEN="你的_DIDA365_ACCESS_TOKEN"

# 2. 示例：查看全量任务树状图
node ./dida365-schedule/scripts/list-dida365-tasks.js

# 3. 示例：创建新任务 (必须提供对应的 projectId)
node ./dida365-schedule/scripts/dida365-task.js "新任务标题" '{"projectId": "YOUR_PROJECT_ID", "priority": 3}'
```
*(更多 API 细节和高级使用方式，请参考 `dida365-schedule/` 目录下的 `SKILL.md` 和 `SKILL_LIST_TASKS.md` 文档)*
