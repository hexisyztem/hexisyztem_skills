# MiAssistant Device Control Skill

This skill provides command-line control for Mi Home (Mijia) smart devices.

## Planned Features

### 1. Device Discovery
List all Mi Home devices in your network with their status and capabilities.

### 2. Device Control
Control individual devices with commands like:
- Turn on/off smart plugs and lights
- Adjust brightness and color temperature
- Control curtain position
- Set air conditioner temperature

### 3. Scene Automation
Trigger predefined Mi Home scenes or create custom automation workflows.

### 4. Schedule Integration
Automatically trigger device actions based on your Dida365 schedule.

## Authentication

Authentication setup is in progress. Will support:
- Mi Home account credentials
- Device tokens for local control
